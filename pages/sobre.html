Recursos utilizados

Neste projeto utilizamos os seguintes recursos:
  • Fizemos um mock inicial da possível aparência do site usando o site canva.com
  • Utilizamos Bootstrap para estilização rápida do site
  • Lendo a documentação do Bootstrap, encontramos o componente carousel e decidimos o usar para a aba "Sustentabilidade"
  • Usamos os lindos ícones do Font Awesome para melhor estilizar o site
  • Fizemos uso de manipulação de HTML e forms com JavaScript para a interação do usuário com o site
